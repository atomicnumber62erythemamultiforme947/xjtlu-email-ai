📚 使用教程

🔑 1. 配置 API

点击页面顶部的 ⚙️ 配置 按钮，填入 AI 服务商的 Base URL、API Key 和模型名称即可。

"ai": { "base_url": "https://api.openai.com/v1", "api_key": "sk-...", "model": "gpt-4o-mini" }

🍪 2. 获取 Cookie

  1. 在 Chrome/Edge 中安装 Cookie-Editor 插件
  2. 登录 mail.xjtlu.edu.cn
  3. 点击 Cookie-Editor → Export as JSON
  4. 在 ⚙️ 配置 中粘贴到 Cookie 字段

提示：Cookie 会定期失效，届时重新导出即可。

📂 3. 文件说明

  • run_app.bat — 双击启动
  • src/config.json — 配置文件
  • src/app.py — 核心服务
